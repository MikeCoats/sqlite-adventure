# SQLite Adventures

## Build

```sh
$ ./build.sh
```

## Play

```sh
$ ./page 1
```

```
Village Limits

You stand at the edge of the forest, a path ahead leads to a small
village. A sign reads “Preas Cnò - pop’n 73”.

If you turn and enter the forest, turn to page 2.
If you head into the village, turn to page 3.
```
